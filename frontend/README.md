# Enterprise RAG RBAC Chatbot

Enterprise AI chatbot using:

- FastAPI
- LangChain
- FAISS
- Groq LLM
- JWT Authentication
- RBAC
- Streamlit UI

- Role-Based Access Control
- Department-specific retrieval
- Retrieval-Augmented Generation
- Source citations
- Monitoring & logging
- Secure API authentication
